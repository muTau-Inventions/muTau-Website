Installation

muTau kann über pip oder aus dem Source installiert werden.

Voraussetzungen

  • Python 3.8 oder höher
  • Vivado HLS 2020.1 oder höher (für Xilinx FPGAs)
  • Intel Quartus Prime (für Intel FPGAs)

Installation via pip

pip install mutau-toolchain

Installation aus Source

git clone https://github.com/mutau-inventions/mutau-toolchain.git
cd mutau-toolchain
pip install -e .

Verifizierung

Überprüfen Sie die Installation mit:

mutau --version

Quick Start

Erste Schritte mit muTau in wenigen Minuten.

1. Modell laden

from mutau import Converter

converter = Converter()
converter.load_model('model.pth', framework='pytorch')

2. In HLS konvertieren

hls_code = converter.to_hls(
    target_fpga='xilinx_zynq',
    optimization_level=2
)

3. SoC Integration

from mutau import SoCBuilder

builder = SoCBuilder()
builder.integrate(hls_code=hls_code, interface='axi')
builder.generate()

from mutau import Converter

converter = Converter()
converter.load_model(model, framework='pytorch')
converter.export('./output/resnet18_hls')
